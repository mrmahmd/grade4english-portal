# Connect Plus Primary 4 – Term 1

A GitHub Pages-ready interactive English learning application.

## Upload
1. Upload **all files and folders** in this directory to the root of your GitHub repository.
2. Keep the paths exactly as they are.
3. In GitHub: **Settings → Pages → Deploy from a branch → main / root**.
4. Open the published link after deployment finishes.

The application stores each student's progress locally in the browser.
